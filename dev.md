Neo4j Tips

1. 查询某节点下游所有节点（不包含该节点）

```
MATCH (n:Task) WHERE n.name="185017" WITH n 
MATCH p = (n) - [*] -> (m:Task)
RETURN m
```

2. 查询某节点上游所有节点（不包含该节点）

```
MATCH (n:Task) WHERE n.name="187509" WITH n 
MATCH p = (m:Task) - [*] ->(n)
RETURN m
```

3. 清空关系和节点
MATCH (n)-[r]-()
DELETE n,r

match (n)
delete n
